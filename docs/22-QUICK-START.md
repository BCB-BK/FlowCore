# Schnellstart – FlowCore

## In 5 Minuten loslegen

### 1. Anmelden
- Öffnen Sie FlowCore über den Link oder den Microsoft Teams-Tab **"FlowCore"**
- Die Anmeldung erfolgt automatisch über Ihr Microsoft-Konto

### 2. Navigieren
- **Seitenleiste** (links): Baumstruktur aller Seiten
- **Suchleiste** (oben): Suche nach Begriffen
- **Meine Aufgaben**: Ihre persönliche Übersicht

### 3. Suchen
- Klicken Sie auf die Suchleiste oder drücken Sie die Tastenkombination
- Geben Sie Ihren Suchbegriff ein
- Ergebnisse werden nach Relevanz sortiert angezeigt
- Klicken Sie auf ein Ergebnis, um die Seite zu öffnen

### 4. Lesen
- Klicken Sie auf eine Seite in der Baumstruktur
- Sie sehen den Inhalt mit Metadaten (Prozess-ID, Status, Tags)
- Verwandte Seiten werden in der Seitenleiste rechts angezeigt

### 5. Bearbeiten (für Editoren)
- Klicken Sie auf **"Bearbeiten"** auf einer Seite
- Nehmen Sie Ihre Änderungen im Block-Editor vor
- Tippen Sie `/` für Slash-Befehle (Tabellen, Bilder, etc.)
- Erstellen Sie eine **neue Revision** zum Speichern

### 6. Teilen
- Klicken Sie auf **"Teilen"** auf einer Seite
- Wählen Sie "Link kopieren" oder "In Teams teilen"

## Häufige Fragen

**Wie finde ich eine bestimmte Seite?**
Nutzen Sie die Suchfunktion oder navigieren Sie über die Baumstruktur.

**Wo sehe ich meine ausstehenden Aufgaben?**
Unter "Meine Aufgaben" in der Navigation.

**Wie erstelle ich eine neue Seite?**
Navigieren Sie zur Elternseite und klicken Sie auf "+ Unterseite".

**Was bedeuten die Prozess-IDs?**
Jede Seite hat eine eindeutige ID (z.B. KP-0001.1), die die Position in der Prozesshierarchie anzeigt.

**Wie funktioniert der Veröffentlichungsprozess?**
Entwurf → Revision erstellen → Review → Genehmigung → Veröffentlichung.
